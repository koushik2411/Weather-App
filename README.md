Project Name: 'Weather App'

Short Description:
Weather App is a project designed using HTML, TailwindCSS and JavaScript. It fetches real-time weather data of a location along with upcoming 5 days weather data. It also have a function to search weather details of your current location without even typing location name, it searches just in a single click of the desired button. It also stores the search history of the cities in the browser local storage. 

Technical Stacks:-
• Frontend: HTML (structure), TailwindCSS (styling and responsiveness)
• Logic: JavaScript (Async/Await for API calls, DOM Manipulation, Event handling)
• Storage: Browser local storage (to store search history)
• Icons: Font Awesome and OpenWeatherMap official Icons
• API: OpenWeatherMap

Key Features:-
• Real-time data: It gives various real-time data such as humidity, wind speed, etc using OpenWeatherMap API.
• Location search: You can manually search weather of a location as well as get weather of your current location.
• Local storage: It keeps a track of search history of cities whose weather is fetched.
• 5-days extended forecast: It fetches upcoming 5 days weather forecast of the location which is searched.
